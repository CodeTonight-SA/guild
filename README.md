# CodeTonight x AI Craftspeople Guild

We're a small South African tech firm that builds tools for human-AI collaboration.
Our main project is **GRIP** -- a system that gives Claude Code persistent memory,
mechanical quality checks, and the ability to learn from its own mistakes.

We joined the AI Craftspeople Guild because we share the same conviction:
AI should be built with craft, not shipped as slop.

## What We Bring

- **GRIP** -- 280 automation recipes, 29 specialised workers, safety rules enforced by
  code (not by asking the AI nicely). Built over 474 development cycles. Open to examination.
- **Ritual 9: Falsify Before You Ship** -- V>> authored this Guild ritual. Every claim
  gets adversarial review before publication. If it doesn't survive, it doesn't ship.
- **Ritual 10: Cost-Aware Retrieval** -- V>> authored this too. Check what you already
  know before burning tokens on expensive searches. Simple idea, large savings.
